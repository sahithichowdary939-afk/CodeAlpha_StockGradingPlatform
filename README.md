# 📈 Stock Trading Platform

## 📌 Overview

The Stock Trading Platform is a Java console-based application that simulates stock market trading operations.

---

## ✨ Features

✅ Display market stock data  
✅ Buy stocks  
✅ Sell stocks  
✅ Portfolio management  
✅ Transaction history  
✅ Balance tracking  
✅ Profit/Loss calculation  
✅ Menu-driven interface  

---

## 🛠 Technologies Used

> Java  
> Object-Oriented Programming  
> Collections Framework  
> ArrayList  
> HashMap  
> File Handling

## ▶️ How to Run

### 🔹 Compile

```bash
javac src/*.java
```

### 🔹 Run

```bash
java src/Main
```
